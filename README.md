# Autofill
랜덤 문자열 자동 입력기

## 👆입력란에 반복작업해야하는 경우
테스트할때 가상의 송장번호나 가상의 비밀번호를 계속 입력할 일이 있는데,
수동으로 10자든 6자든 계속 달리 누르는게 번거롭기 때문에 클릭 3번으로 해결하고자 함.

## 👆실행방법
1. exe파일 실행하면 작업표시줄 우측 ^표시(시스템트레이)에 랜덤입력기 아이콘 생김

2. 우클릭하여 옵션 설정으로 들어가 원하는 문자열 조합을 설정하고 저장

3. 작업표시줄에 아이콘을 한번 클릭 후
4. 웹페이지나 어디든 클릭 > 정말 입력할 곳에 클릭(즉 2번째 클릭 영역)하면 미리 설정된 조합의 랜덤 문자가 입력됨
